# News App

## Overview

A simple news application built using HTML, CSS, and JavaScript (Vanilla JS). The app fetches data from News API and displays it dynamically without reloading the page, following the SPA concept.

---

## Features

* Display Top Headlines
* Display Sources
* Responsive design using Flexbox and Grid
* Sidebar navigation
* Dynamic content rendering without page reload
* Open news articles in a new tab
* Basic error handling
* Fallback images for missing data

---

## Technologies Used

* HTML5
* CSS3 (Flexbox and Grid)
* JavaScript (ES6+)
* Fetch API
* Async / Await

---

## Project Structure

* index.html → Main page
* style.css → Styling
* main.js → Logic and API handling
* assets → Images (optional)

---

## How It Works

1. On page load, the app fetches top headlines
2. When a sidebar link is clicked:

   * The active link is updated
   * Data is fetched from the API
   * Previous content is cleared
   * New content is rendered

---

## API Used

* News API
* Endpoints:

  * Top Headlines
  * Sources
